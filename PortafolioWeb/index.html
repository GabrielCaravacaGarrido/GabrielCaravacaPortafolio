Viajeros Solos

La creación de una página web enfocada en el sector turístico fue llevada a cabo con un enfoque predominantemente en frontend, con una mínima utilización de backend. La base de datos fue gestionada mediante MongoDB para la administración de usuarios. La gestión de versiones se realizó a través de GitHub y la administración del equipo se coordinó mediante Asana.

Se formó un equipo denominado Darkcode, con su página correspondiente, y fui designado como coordinador del mismo. La página web ofrecía diversas funcionalidades, incluyendo roles (usuario, administrador, dueño de servicio), sistema de inicio de sesión, reserva de servicios (hoteles, transporte, comida, entretenimiento), reseñas y pagos.

The creation of a website focused on the tourism sector was carried out with a predominantly frontend approach, with minimal use of backend. The database was managed using MongoDB for user administration. Version control was handled through GitHub, and team management was coordinated via Asana.

A team named Darkcode was formed, with its corresponding webpage, and I was appointed as its coordinator. The website offered various functionalities, including roles (user, administrator, service owner), login system, service reservations (hotels, transportation, food, entertainment), reviews, and payments.

Tu Salud Primero

Creación de una página web enfocada en el sector de medicina. A diferencia de Viajeros Solos, esta página fue realizada con un backend más extenso en C#, junto a una base de datos en Microsoft Azure. Además, el manejo de versiones y la administración del equipo también se realizó en Azure.

Mi puesto en este proyecto fue el de QA teniendo que dar retroalimentación al trabajo de mis compañeros asegurando un producto ordenado, atractivo y funcional. La página ofrecía roles (pacientes, doctores, enfermeros, administradores), sistema de inicio de sesión, agenda de citas, perfiles personalizables, historiales médicos, recetas médicas, sistema de pagos, reportes administrativos, entre otros.

Creation of a website focused on the medical sector. Unlike "Viajeros Solos," this website was developed with a more extensive backend in C#, along with a database in Microsoft Azure. Additionally, version control and team management were also handled in Azure.

My role in this project was as QA, where I provided feedback on my colleagues' work, ensuring an organized, attractive, and functional product. The website offered roles (patients, doctors, nurses, administrators), login system, appointment scheduling, customizable profiles, medical histories, prescriptions, payment system, administrative reports, among other features.
